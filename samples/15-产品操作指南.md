## 第一章 系统概述

### 1.1 系统简介

本系统是基于大语言模型的**企业级私有化知识检索与问答平台**，帮助企业把分散在各处（文档、邮件、Wiki、工单等）的知识资产集中管理，并通过自然语言问答的方式向员工提供精准、可信的知识服务。

### 1.2 核心能力

- **多模态文档解析**：支持 PDF、Word、Excel、PPT、图片、音视频等 100+ 格式
- **混合检索**：结合关键词匹配和向量语义匹配，准确率高达 95%+
- **智能问答**：基于大语言模型生成自然语言回答，每条回答附原文引用
- **4 级权限**：文件类型 / 文档 / 字段 / 标签级别的精细化权限控制
- **审计追溯**：完整记录所有操作，支持合规审计
- **API 集成**：提供 RESTful API 和外部 API Key，方便与企业现有系统对接

### 1.3 适用场景

- 企业内部知识库问答
- 客户服务辅助
- 研发知识管理
- 培训与学习辅助
- 合规与法务咨询
- 产品手册查询

## 第二章 管理员操作手册

### 2.1 用户管理

#### 2.1.1 创建用户

1. 进入 **系统管理 → 用户管理**
2. 点击「+ 新建用户」按钮
3. 填写用户信息：
   - 用户名（必填，唯一）
   - 邮箱（必填，唯一）
   - 姓名
   - 部门
   - 角色（管理员 / 部门管理员 / 普通用户）
   - 安全等级（L0–L4）
4. 点击「保存」，系统自动发送初始密码邮件

#### 2.1.2 用户角色说明

| 角色 | 权限范围 |
|---|---|
| 系统管理员 | 全部功能，包括用户管理、系统配置、审计日志 |
| 部门管理员 | 本部门用户管理、知识库管理 |
| 普通用户 | 使用检索、问答、查看已授权知识库 |

#### 2.1.3 重置密码

1. 在用户列表找到目标用户
2. 点击「重置密码」
3. 系统生成新密码并显示，**请记录后告知用户**
4. 用户首次登录后必须修改密码

### 2.2 知识库管理

#### 2.2.1 创建知识库

1. 进入 **知识库 → 我的知识库**
2. 点击「+ 新建知识库」
3. 填写：
   - 知识库名称（必填）
   - 描述
   - 可见范围（公开 / 部门 / 指定用户）
4. 点击「保存」

#### 2.2.2 上传文档

支持的批量上传方式：

- **方式 1**：拖拽文件到上传区
- **方式 2**：点击「上传」按钮选择文件
- **方式 3**：通过 API 上传

文件大小限制：
- 单个文件：≤ 500 MB
- 批量上传：每次 ≤ 50 个文件

#### 2.2.3 文档处理流程

```
上传 → 解析 → 切分 → 嵌入向量 → 存储
 ↓     ↓      ↓       ↓         ↓
存储  文本  Chunk  Embedding  pgvector
```

文档状态说明：

- **pending**：待处理
- **processing**：正在解析
- **indexed**：已索引，可检索
- **failed**：处理失败，可在详情中查看错误日志

#### 2.2.4 文档权限设置

1. 在知识库中选中目标文档
2. 点击「权限设置」
3. 选择权限级别：

| 级别 | 配置方式 |
|---|---|
| L1 文件类型 | 整个文件类型（如 PDF）只对某些用户可见 |
| L2 文档 | 单个文档对指定用户/部门/群组可见 |
| L3 字段 | Excel 文档的某些列只对特定用户可见 |
| L4 标签 | 带特定标签的文档只对特定用户可见 |

### 2.3 API Key 管理

#### 2.3.1 创建 API Key

1. 进入 **系统管理 → API Key 管理**
2. 点击「+ 新建 API Key」
3. 配置：
   - 名称
   - 作用域（scopes）：search / chat / kb:read / doc:write 等
   - 速率限制（RPM）
   - 过期时间（可选）
4. **重要**：保存时系统只显示一次明文 Key，请妥善保管

#### 2.3.2 可用作用域

| 作用域 | 说明 |
|---|---|
| search | 检索接口 |
| chat | 问答接口 |
| kb:read | 读取知识库列表 |
| kb:write | 创建/修改知识库 |
| doc:read | 读取文档 |
| doc:write | 上传/删除文档 |
| user:read | 读取用户信息 |

#### 2.3.3 调用示例

```bash
# 用 API Key 调用搜索接口
curl -X POST https://your-domain/api/v1/external/search \
  -H "X-API-Key: rag_live_xxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "年假如何申请",
    "kb_ids": ["kb-uuid"],
    "top_k": 5
  }'
```

### 2.4 敏感词管理

#### 2.4.1 添加敏感词

1. 进入 **系统管理 → 敏感词管理**
2. 点击「+ 添加」
3. 配置：
   - 关键词
   - 等级（L1-L5）
   - 类别（个人信息 / 商业机密 / 法律法规 / 其他）
   - 匹配方式（精确 / 模糊 / 正则）
   - 处理方式（脱敏 / 拦截 / 告警）

#### 2.4.2 批量导入

支持 Excel 模板批量导入。模板包含：

| 关键词 | 等级 | 类别 | 匹配方式 | 处理方式 |
|---|---|---|---|---|
| 身份证号 | L5 | 个人信息 | 正则 | 脱敏 |
| 公司机密 | L4 | 商业机密 | 模糊 | 拦截 |

### 2.5 运营监控

#### 2.5.1 仪表盘

进入 **运营数据** 查看：

- 今日问答数
- 今日检索数
- 活跃用户数
- 系统响应时间
- 错误率

#### 2.5.2 审计日志

1. 进入 **运营数据 → 审计日志**
2. 可按时间、用户、操作类型筛选
3. 支持导出 CSV / Excel
4. 日志保留期：180 天（可配置）

## 第三章 业务用户操作手册

### 3.1 登录系统

1. 打开浏览器访问 https://your-domain
2. 输入用户名和密码
3. 首次登录需要修改初始密码

### 3.2 检索问答

#### 3.2.1 简单搜索

1. 进入 **搜索控制台**
2. 在搜索框输入问题，例如：
   - "公司年假如何申请？"
   - "财务报销需要哪些材料？"
   - "XX 产品的保修期是多久？"
3. 选择目标知识库（可多选）
4. 点击「搜索」或回车

#### 3.2.2 查看结果

搜索结果列表展示：
- 文档标题
- 相关度分数（百分比）
- 内容片段
- 来源信息

点击「查看原文」可打开文档查看完整内容。

#### 3.2.3 智能问答

1. 在搜索结果页面点击「智能问答」
2. 系统基于检索结果自动生成回答
3. 每条回答下方展示「引用来源」
4. 可点击来源跳转到原文段落

### 3.3 多轮对话

1. 进入 **聊天** 模块
2. 选择目标知识库
3. 输入第一个问题
4. 后续问题可基于上下文连续对话
5. 可点击「新建会话」开始新话题

### 3.4 协作功能

#### 3.4.1 评论

在任何文档页面，可以选中段落添加评论：

1. 选中要评论的文本
2. 点击悬浮的「+ 评论」按钮
3. 输入评论内容
4. 评论会通知相关用户

#### 3.4.2 收藏

1. 在文档详情页点击「收藏」
2. 收藏的文档可在「我的收藏」中查看
3. 支持文件夹分类管理

## 第四章 常见问题

### Q1：搜索结果为空怎么办？

A：请尝试：

1. 简化搜索关键词
2. 检查关键词拼写
3. 选择更广泛的知识库
4. 切换到「关键词搜索」或「语义搜索」

### Q2：智能问答回答不准确怎么办？

A：可尝试：

1. 明确问题范围，限定知识库
2. 增加搜索结果数量（top_k）
3. 提供更多上下文
4. 反馈错误（点击回答下方的「反馈」按钮）

### Q3：上传文档后多久能搜索到？

A：根据文档大小：

- 小文档（< 10 页）：立即
- 中等文档（10-100 页）：1-5 分钟
- 大文档（> 100 页）：5-30 分钟

可在「文档管理」查看处理状态。

### Q4：忘记密码怎么办？

A：请联系系统管理员重置。

### Q5：账号被锁定怎么办？

A：连续 5 次输错密码会锁定账号，30 分钟后自动解锁，或联系管理员立即解锁。
