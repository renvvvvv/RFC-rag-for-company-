## 第一条 合同标的

### 1.1 产品名称

**企业级私有化多模态 RAG 知识库系统 V2026**（以下简称"标的软件"）。

### 1.2 产品组成

标的软件包括以下组成部分：

1. 核心平台软件（FastAPI + React 全栈）
2. 后台管理模块（用户、权限、知识库、文档管理）
3. 检索增强生成（RAG）引擎
4. 多模态文档解析模块（PDF、Word、Excel、图片、音视频）
5. 4 级权限控制系统（L1 文件类型 / L2 文档 / L3 字段 / L4 标签）
6. 敏感信息识别与脱敏模块
7. 运营管理与审计模块
8. 外部 API 网关（Kong + JWT + API Key 鉴权）
9. 监控告警模块（Prometheus + Grafana + AlertManager）
10. 部署运维工具链（Docker / Helm Charts）

### 1.3 部署方式

甲方选择以下部署方式：

- ☐ **私有化部署**：部署在甲方自有数据中心
- ☐ **专有云部署**：部署在甲方指定的云服务商
- ☐ **混合云部署**：核心数据本地，AI 推理云端

## 第四条 交付与验收

### 4.1 交付时间

乙方应在合同生效后 **30 个工作日内** 完成系统部署。

### 4.2 交付物

1. 标的软件安装包及镜像
2. 系统部署文档
3. 用户操作手册
4. API 接口文档
5. 运维管理手册
6. 测试报告
7. 培训服务（详见第七条）

### 4.3 验收标准

系统应满足：

- 所有功能模块按《产品功能说明书》实现
- P95 接口响应时间 < 1 秒
- 系统可用性 ≥ 99.9%
- 安全漏洞扫描通过
- 性能压测通过（详见技术规格书）
- 甲方指定 5 个核心业务场景验证通过

### 4.4 验收程序

1. 乙方提交验收申请
2. 甲方在 7 个工作日内组织验收测试
3. 验收通过 → 签署《验收报告》
4. 验收不通过 → 乙方在 10 个工作日内整改

### 5.2 质保范围

3. 7×24 小时故障响应
4. 工作日工作时间电话支持

| 故障级别 | 描述 | 响应时间 | 解决时间 |

### 6.2 客户数据

### 6.3 二次开发

## 第八条 保密条款

### 8.1 保密信息

双方在合同履行期间知悉的对方技术、商业、客户、人员等信息均属保密信息。

### 8.2 保密期限

保密义务自合同签订之日起持续 **5 年**，无论合同是否履行完毕。

### 8.3 违约责任

违反保密义务的，违约方应向守约方支付违约金人民币 **¥500,000 元**，造成损失的另行赔偿。

## 第九条 违约责任

### 9.1 乙方违约

1. 未按期交付：每延误一天，按合同总额 0.5‰ 支付违约金
2. 验收不合格：整改超过 3 次仍未通过，扣除合同尾款
3. 故障响应超时：每次扣除当月运维费的 5%

### 9.2 甲方违约

1. 未按期付款：每延误一天，按未付金额 0.5‰ 支付违约金
2. 擅自修改、转让软件：乙方有权解除合同并追究法律责任

## 第十条 不可抗力

### 10.1 不可抗力定义

指不能预见、不能避免且不能克服的客观情况，包括但不限于：

- 自然灾害（地震、洪水、台风等）
- 战争、军事行动、恐怖活动
- 重大疫情、政府行为
- 大规模网络攻击、电力中断

### 10.2 处理

发生不可抗力时，受影响方应在 3 日内书面通知对方，并在 30 日内提供有效证明，双方协商处理。

## 第十一条 争议解决

### 11.1 协商优先

合同履行过程中发生的争议，双方应首先通过友好协商解决。

### 11.2 诉讼管辖

协商不成的，任一方可向**乙方所在地有管辖权的人民法院**提起诉讼。

## 第十二条 其他约定

### 12.1 合同生效

本合同自双方法定代表人或授权代表签字盖章之日起生效。

### 12.2 合同变更

### 12.3 合同份数
