## 第 2 页：产品定位与价值

### 页面内容
- 一句话定位：面向中大型企业的私有化多模态知识库问答平台。
- 三大核心价值：
  1. **知识资产统一入口**：文档、Excel、图片、视频、链接一站式管理。
  2. **安全合规**：五级权限穿透 + 审计日志，满足金融、政务、制造合规要求。
  3. **降本增效**：自然语言问答替代人工检索，平均检索耗时从 10 分钟降至 30 秒。

### 演讲者备注
- 用客户实际场景举例：新员工查制度、研发查技术规范、销售查产品资料。
- 强调私有化与 SaaS RAG 的本质差异：数据主权、模型可控、权限可定制。

---

## 第 3 页：系统架构与核心能力

### 页面内容
- 四层架构图：应用层 → 网关层 → 服务层 → 数据层。
- 核心组件：
  - 前端：React + Ant Design（Web / 管理后台 / 评测工作台）
  - 网关：Kong（限流、路由、负载均衡）
  - 后端：FastAPI（检索、生成、权限、评测）
  - 存储：PostgreSQL + Milvus + MinIO + Redis + RabbitMQ
  - 监控：Prometheus + Grafana

### 演讲者备注
- 解释“统一 API 入口”的意义：外部业务系统、Agent、IM 机器人可快速集成。
- 强调 Pipeline 可编排：不同模态独立处理，但共享统一检索与生成服务。

---

## 第 4 页：多模态知识库管理

### 页面内容
- 支持的文件类型：PDF / Word / Excel / PPT / 图片 / 视频 / 网页链接。
- 处理流程：上传 → 解析 → 分块/切片 → 向量化 → 索引 → 可检索。
- 文档状态看板：待解析 / 解析中 / 已索引 / 解析失败 / 已禁用。
- 批量上传与链接自动更新机制。

### 演讲者备注
- 演示上传 1 个 PDF 和 1 个 Excel 文件，展示解析进度与索引完成提示。
- 提醒视频文件解析耗时较长，建议后台异步处理。

---

## 第 5 页：搜索与问答体验

### 页面内容
- 全局搜索：跨知识库、跨模态返回 Top-K 结果。
- 单库问答：基于指定知识库生成答案并引用来源。
- 答案溯源：文本段落、Excel 单元格、图片、视频片段均可定位。
- 多轮对话：保留上下文，支持连续追问。

### 演讲者备注
- 现场演示一个业务问题，例如“2025 年 Q1 产品发布计划是什么？”
- 强调引用溯源是建立用户信任的关键。

---

## 第 6 页：五级权限与安全管理

### 页面内容
- 五级权限穿透：文件类型 → 文档 → 字段/段落 → 标签 → 关键词。
- RBAC + ABAC 混合模型：角色 + 属性（部门、密级、项目）双重控制。
- 权限申请与审批流程。
- 审计日志：登录、上传、检索、权限变更全程留痕。

### 演讲者备注
- 用“财务 Excel 表中的敏感列仅对财务组可见”举例说明字段级权限。
- 说明关键词拦截如何防止生成内容中出现敏感信息。

## 第 7 页：典型应用场景

### 页面内容
- **场景 1：研发知识库**——技术方案、接口文档、发布说明统一检索。
- **场景 2：销售赋能**——产品资料、竞品分析、客户案例快速问答。
- **场景 3：合规审计**——制度文件、合同、审计报告权限化访问。
- **场景 4：培训学习**——视频课程切片检索，员工按需学习。

### 演讲者备注
- 邀请听众思考自己部门最适用的场景。
- 强调“一个系统可以支撑多个业务域，只要权限隔离到位”。

---

## 第 8 页：最佳实践与 FAQ

### 页面内容
- 最佳实践：
  1. 知识库命名规范：业务域 + 用途 + 版本。
  2. 标签体系提前规划，便于后续权限与检索过滤。
  3. 重要文档设置密级与过期时间。
  4. 定期使用评测工作台验证检索效果。
- FAQ 精选（3~4 个高频问题）：
  - 上传后多久可以搜索？
  - 为什么有些文档搜不到？
  - 如何申请访问权限？
  - 模型配置修改后是否需要重启？

### 演讲者备注
- 总结培训要点，发放用户手册与 FAQ 文档。
- 预留 10 分钟答疑，记录高频问题补充至 FAQ。
