# 企业级私有化多模态 RAG 系统 — 用户操作手册

> 版本：v1.0  
> 更新日期：2026-06-15  
> 适用对象：KB 管理员、普通用户、访客  

---

## 1. 手册概述

本文档面向最终用户，介绍企业级私有化多模态 RAG 知识库系统的日常使用流程，包括登录、知识库创建、文档上传、检索/问答、权限申请、模型配置等核心操作。

## 2. 系统登录

### 2.1 访问入口

在浏览器地址栏输入系统域名，例如：

```
https://rag.company.com
```

> 私有化部署场景下，请使用企业分配的域名或内网 IP。

### 2.2 账号密码登录

1. 进入登录页。
2. 输入企业邮箱（或用户名）和密码。
3. 点击 **登录**。

`[截图占位：登录页]`

### 2.3 SSO / LDAP 登录（如已配置）

1. 点击 **企业微信登录** / **飞书登录** / **LDAP 登录**。
2. 按企业身份 provider 引导完成授权。
3. 首次登录后，系统自动创建默认权限的账号。

### 2.4 常见问题

| 现象 | 处理方式 |
|------|----------|
| 忘记密码 | 联系 KB 管理员或 Super Admin 重置 |
| 提示“账号无权限” | 确认是否已被加入目标知识库或用户组 |
| 二维码登录失败 | 检查企业 IM 授权配置及网络连通性 |

## 3. 创建知识库

### 3.1 进入知识库管理

1. 登录后，点击顶部导航 **知识库管理**。
2. 点击右上角 **新建知识库**。

`[截图占位：知识库列表页]`

### 3.2 填写知识库信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 知识库名称 | 必填，建议体现业务归属 | 研发技术规范库 |
| 描述 | 可选，说明知识库用途 | 存放产品技术方案、接口规范、发布说明 |
| 可见范围 | 私有 / 部门可见 / 公开 | 部门可见 |
| 默认权限 | 上传、下载、检索、问答 | 仅检索 |

### 3.3 选择模态与处理策略

系统支持以下模态：

- 文档：PDF、Word、PPT、TXT、Markdown
- 表格：Excel、CSV
- 图片：PNG、JPG、WebP
- 视频：MP4、MOV、AVI（自动切片+关键帧提取）
- 链接：网页 URL（自动抓取正文）

`[截图占位：新建知识库弹窗]`

### 3.4 保存并进入知识库

点击 **创建**，系统会自动跳转至该知识库的文档管理页。

## 4. 上传与管理文档

### 4.1 单文件上传

1. 进入目标知识库，点击 **上传文档**。
2. 选择本地文件或拖拽文件至上传区域。
3. 系统自动识别文件类型并显示解析进度。

`[截图占位：上传中心]`

### 4.2 批量上传

1. 点击 **批量上传**。
2. 一次性选择多个文件（建议单次不超过 100 个）。
3. 系统按队列异步解析，可在 **任务中心** 查看进度。

### 4.3 链接录入

1. 点击 **添加链接**。
2. 输入网页 URL，选择是否自动更新。
3. 系统将抓取页面正文、图片 alt 文本，并建立索引。

### 4.4 文档状态说明

| 状态 | 含义 |
|------|------|
| 待解析 | 文件已上传，等待 Celery Worker 处理 |
| 解析中 | 正在执行 OCR、分块、向量化 |
| 已索引 | 已完成，可参与检索 |
| 解析失败 | 查看失败原因，可点击 **重新处理** |
| 已禁用 | 文档不参与检索，可重新启用 |

`[截图占位：文档列表及状态]`

### 4.5 文档元数据编辑

点击文档行末的 **编辑** 按钮，可修改：

- 文档标签（用于权限与过滤）
- 密级（公开 / 内部 / 机密 / 绝密）
- 过期时间
- 自定义字段

## 5. 搜索与问答

### 5.1 全局搜索

1. 在顶部搜索框输入自然语言问题或关键词。
2. 按回车，系统跨知识库、跨模态返回 Top-K 结果。
3. 左侧筛选栏可按模态、知识库、时间范围、标签过滤。

`[截图占位：全局搜索结果页]`

### 5.2 单知识库问答

1. 进入目标知识库。
2. 在对话输入框输入问题。
3. 系统基于该知识库内容生成答案，并展示引用来源。

`[截图占位：单库问答页]`

### 5.3 答案引用与溯源

- 答案下方会列出引用的文档片段（Chunk）。
- 点击 **查看原文** 可跳转至对应文档位置。
- 支持多模态引用：文本段落、Excel 单元格、图片、视频片段。

### 5.4 多轮对话

在问答模式下，系统默认保留当前会话上下文。点击 **新会话** 可清空上下文。

### 5.5 反馈

对每次回答，可点击 **点赞** / **点踩**，帮助管理员优化模型与检索策略。

## 6. 权限管理（面向 KB 管理员）

### 6.1 权限模型概览

系统支持五级权限穿透：

```
文件类型 → 文档 → 字段/段落 → 标签 → 关键词
```

并支持 RBAC（角色）+ ABAC（属性）混合授权。

### 6.2 给用户授权

1. 进入 **权限管理** 页面。
2. 选择目标知识库或文档。
3. 点击 **添加成员**，选择用户或用户组。
4. 配置权限：查看、检索、上传、下载、管理。

`[截图占位：权限管理页]`

### 6.3 字段级权限

针对 Word / Excel 文档，可设置段落、单元格、工作表级别的可见性：

1. 在文档详情页点击 **字段权限**。
2. 选择需要控制的范围。
3. 绑定用户组或密级条件。

### 6.4 标签级权限

1. 进入 **标签管理**。
2. 创建标签并绑定权限规则。
3. 在上传或编辑文档时打标签，权限自动生效。

### 6.5 权限申请与审批

普通用户可针对无权限的知识库发起 **申请访问**：

1. 在知识库列表点击 **申请权限**。
2. 填写申请理由。
3. KB 管理员在消息中心审批。

## 7. 模型配置（面向 Super Admin）

### 7.1 进入模型配置

1. 点击右上角头像 → **系统管理** → **模型配置**。

`[截图占位：模型配置页]`

### 7.2 配置 Embedding 模型

| 字段 | 说明 |
|------|------|
| 模型名称 | 便于识别的名称 |
| 服务地址 | Embedding 服务 URL，如 `http://embedding-service:8080` |
| 向量维度 | 与向量库一致，如 1024 |
| 是否默认 | 是否作为新建知识库的默认模型 |

### 7.3 配置 Re-rank 模型

| 字段 | 说明 |
|------|------|
| 模型名称 | 便于识别的名称 |
| 服务地址 | Re-rank 服务 URL，如 `http://rerank-service:8080` |
| Top-K 重排 | 对召回结果重排序的条数 |

### 7.4 配置生成模型

| 字段 | 说明 |
|------|------|
| Provider | minimax / OpenAI / 私有模型 |
| API Key | 模型服务密钥 |
| Base URL | API 地址 |
| 默认温度 | 控制生成随机性，建议 0.3~0.7 |
| 最大 Token | 单次生成上限 |

> 配置保存后即时生效，无需重启后端服务。

## 8. 评测工作台（面向管理员）

### 8.1 创建评测数据集

1. 进入 **评测工作台**。
2. 点击 **新建数据集**，导入问答对（支持 JSON / Excel）。

### 8.2 发起评测任务

1. 选择评测数据集与目标知识库。
2. 选择评估指标：Recall@K、MRR、NDCG、Faithfulness、Relevance。
3. 点击 **开始评测**。

### 8.3 查看报告

系统生成可视化报告，支持 A/B 对比不同模型或参数配置。

`[截图占位：评测工作台]`

## 9. 审计与日志

### 9.1 查看个人检索历史

在 **我的** → **检索历史** 中查看自己曾经的问题与答案。

### 9.2 管理员审计

Super Admin 可在 **系统管理** → **审计日志** 中查看：

- 用户登录/登出
- 文档上传/下载
- 权限变更
- 检索与问答记录（可配置是否记录原始问题）

## 10. 常见问题速查

| 问题 | 解决方案 |
|------|----------|
| 上传后搜不到 | 检查文档状态是否为“已索引”，密级是否有权限 |
| 答案不准确 | 尝试调整检索 Top-K、Re-rank 模型或知识库边界 |
| 大视频解析慢 | 视频切片与向量化耗时较长，可在任务中心查看进度 |
| 想导出问答记录 | 联系管理员，通过审计日志导出 |
