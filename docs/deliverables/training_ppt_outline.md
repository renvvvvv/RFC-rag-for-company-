# 企业级私有化多模态 RAG 系统培训 PPT 大纲

> 适用对象：企业知识管理员、普通员工、系统管理员
> 培训时长：约 60 分钟（可根据听众拆分）

---

## 第 1 页：封面

- 标题：企业级私有化多模态 RAG 系统使用培训
- 副标题：安全可控的企业知识助手
- 版本与日期

## 第 2 页：培训目标

- 理解系统核心能力（知识库、检索、问答、权限）
- 掌握文档上传与知识库管理
- 学会使用检索与多轮问答
- 了解安全等级与敏感信息控制

## 第 3 页：系统定位与价值

- 私有化部署：数据不出域
- 多模态支持：文本 / 表格 / 图片 / 音频 / 视频
- 权限五级穿透：L0-L4 安全等级
- 典型场景：合规检索、研发知识库、客服助手、内部培训

## 第 4 页：系统架构概览

- 前端：React + Vite + Ant Design
- 后端：FastAPI + PostgreSQL(pgvector) + Redis + Celery
- 可选：Milvus + MinIO + RabbitMQ + Kong（全量版）
- 轻量版：4 容器、< 1GB 内存即可运行

## 第 5 页：角色与权限（L0-L4）

- L0：普通查询者
- L1：可上传文档
- L2：可创建知识库
- L3：用户/组/API Key 管理
- L4：系统级配置与脱敏策略

## 第 6 页：登录与首页

- 账号密码登录
- 多语言切换（i18n）
- 首页功能入口：知识库、检索、问答、权限、系统配置

## 第 7 页：知识库管理

- 创建知识库：名称、描述、配置
- 查看统计：文档数、分块数、状态分布
- 更新与删除知识库
- 实践演示：创建“产品研发知识库”

## 第 8 页：文档上传与解析

- 支持的格式：PDF、Word、Excel、PPT、图片、音频、视频、链接
- 上传流程：选择文件 → 标签 → 后台解析 → 分块向量化
- 文档状态：pending / processing / completed / failed
- 重新处理（reprocess）

## 第 9 页：检索控制台

- 混合检索：向量 + BM25 + RRF + Cross-Encoder 重排序
- 三种模式：hybrid / semantic / keyword
- 结果展示：chunk 内容、来源文档、相似度、关键词等级
- 搜索历史

## 第 10 页：多轮问答

- 新建会话 / 选择会话
- 关联知识库
- 流式与非流式回答
- 答案引用溯源：doc_id、chunk_id、position_info
- 消息反馈：点赞 / 点踩

## 第 11 页：权限与敏感信息

- 五级权限穿透：文件类型 → 文档 → 字段 → 标签 → 关键词
- 批量授权与继承校验
- 敏感关键词扫描与脱敏
- 安全网关：提示注入检测、外部 API 策略控制

## 第 12 页：API Key 与外部集成

- 创建 API Key：选择 scope、设置限速与过期时间
- 外部 API 认证：X-API-Key / Bearer Token
- 可用外部接口：搜索、问答、知识库、文档管理
- 典型集成：企业内部系统调用

## 第 13 页：评测与运营

- 评测数据集与任务
- 指标：Recall@K、MRR、NDCG、Faithfulness、Relevance
- 监控看板：Grafana + Prometheus
- 告警通道：Webhook / Email / Slack / 企业微信

## 第 14 页：常见问题与最佳实践

- 文档解析失败怎么办？
- 如何提高检索准确率？
- 权限冲突如何排查？
- 模型服务如何配置？
- 推荐：先建小规模测试知识库，再逐步扩大

## 第 15 页：资源与支持

- 用户手册、FAQ、API 使用指南
- 上线 Checklist
- 技术支持渠道
- Q&A
